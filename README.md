# PAC-Bear-Game
A quick demo web app game based on the classic maze game PAC-MAN. Except now we center a bear!

==========================================================================================

This is an output of the WGU AWS Game Dev Event! Most of the heavy lifting (coding) was done by AWS Kiro (Gen AI). The free trial credits ran out quickly, so I have to do the rest of the work. Since the bigger picture has been completed by Kiro, what I have done is to fine tune some parts of the code and some testing to make sure the game app running my way. 


Special thanks to AWS solutions team to hosting the event! 

The project is live at: https://pac-bear-game.onrender.com
